機能
itpManagerは、株式会社アイティプランツが提供する、室内専用植物栽培装置アイティプランターとパソコンをUSBで接続して、クラウド・ガーデンにてコントロールするためのソフトウエアです。

itpManagerの特徴
1.	タスクトレイに常駐して動作するので、作業の邪魔になりません。
2.	アイティプランターの全ての機能をクラウド・ガーデンからコントロールできます。
3.	アイティプランターに内蔵されているセンサーの値をモニターすることができます。
4.	栽培プログラムをアイティプランターに送信できるようになります。
5.	パソコンにUSBカメラを接続すれば、自動で栽培状況を撮影して、Google社のPicasa　webアルバムにアップロードできます。
6.	水切れや温度異常、照度異常がメールで送られてきます。知らない内に水不足で植物を枯らすことが少なくなります。
7.	複数のアイティプランターをコントロールできます。
8.	Javaアプリケーションで作られているので、Windowsの他にもMacOSXでも動作します。

さあ、itpManagerを使って、クラウド・ガーデンライフをお楽しみください。
